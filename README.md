# Python Expense Tracker

A simple command-line expense tracker written in Python.

## Features
- Add expenses with category, amount, and description
- Automatically assigns unique IDs and timestamps
- View total expenses grouped by category
- Stores data in a CSV file (`expenses.csv`)

## Technologies
- Python 3.x
- `csv` module
- `datetime`
- Command-line interface

## Sample usage

```bash
python tracker.py
